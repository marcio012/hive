# 🐝 HIVE OS

O Hive é um Framework de Inteligência Simbiótica para orquestração de desenvolvimento de software multi-agente.

## 🏗️ Estrutura Soberana
- **`beehive/`**: O Núcleo (Core). Contém a inteligência, regras de negócio da colmeia e o motor operacional.
- **`produto/`**: A Obra. Onde os produtos gerados pelo Hive (ex: TenantOS) residem.
- **`scripts/`**: Utilitários de sistema.

## 🧠 Operação (Squad)
Este repositório é operado por um squad de IAs especializadas:
- **Gemini (Tech Lead)**: Orquestração e Veto.
- **Claude (Arquiteto)**: Design e Estratégia.
- **Copilot (Engenheiro)**: Implementação e Execução.

## 🚀 Como Iniciar
Para interagir com a colmeia, utilize os comandos do framework (via `beehive/bin/` ou `package.json`).
