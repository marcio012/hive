# IA no Projeto

Area dedicada a ativos de IA, separada por finalidade para reduzir confusao entre operacao interna e produto.

## Estrutura
- `construcao/`: IA usada para construir, operar e governar o projeto.
- `produto/`: IA usada como funcionalidade do produto para tenants e usuarios finais.

## Objetivo
Garantir rastreabilidade de decisoes e acelerar execucao com limites claros entre:
- IA para o time de engenharia/operacao.
- IA para gerar valor no ERP white label.

## Atalhos
- Construcao: `ai/construcao/README.md`
- Produto: `ai/produto/README.md`

## Regra pratica
Toda preferencia operacional recorrente deve ser registrada em arquivo versionado e revisavel.
