## Estrutura de Governança V2
O Squad opera sob um modelo de **Liderança Operacional Centralizada**:
- **Gemini**: Squad Lead / Integrador (Orquestração e Síntese)
- **Claude**: Arquiteto / Estrategista (Design e Tese)
- **Copilot**: Engenheiro de Software (Execução e Validação)
