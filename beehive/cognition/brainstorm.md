# 🚀 O Processo de Brainstorming (Ideação) no HIVE

## 1. O que é?
O Brainstorming no HIVE é a fase de **Incubação**. É onde o sistema atua como uma "Parede Branca" para o Owner descarregar ideias. Não é um chat comum; é um processo de filtragem para garantir que apenas o que tem valor real chegue à fase de construção.

## 2. Objetivo
Transformar o "eu quero" do Márcio em um "isso resolve esta dor e escala assim". O objetivo final é a **purificação da intenção**.

## 3. Dinâmica
- **Higiene Cognitiva:** O sistema deve "resetar" o contexto técnico (código) para focar apenas no negócio.
- **Simbiose:** O Owner entra com a Intenção, o HIVE entra com a Orquestração e a Provocação.

## 4. Definição de Pronto (Done)
A fase termina quando há um consenso sobre o valor e um artefato de saída é gerado na `collective_intelligence`.
